A Simple Kernel

Bochs IA-32 (80x86) simulated kernel using C and X86 assembly code
November 2019

To understand how operating systems are built, I implemented a basic kernel with a partner as a part of the CPSC 415 course.

I used a Bochs IA-32 (80x86) emulator to run the kernel. Starting from a small program that bootstraps into a basic kernel, I added the following functionality:

  • Basic memory management
  • Process management
    • Time-sharing and process preemption
    • A sleep device
  • Inter-process communication
    • Message passing
    • Signal handling
  • Device interactions
    • A keyboard device
    • The random and zero virtual devices
  • A simple shell that accepts a limited number of commands
